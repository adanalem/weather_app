# responsive_weather_app
A modern, responsive weather application built with a focus on UI/UX, featuring a sleek glassmorphism design. This app provides real-time weather updates and a 24-hour hourly forecast.
Key Features
Live Weather Data: Fetches real-time temperature, humidity, wind speed, and weather conditions for any city.

Hourly Forecast: Displays the next 24 hours of weather data to keep you prepared.

Dynamic Backgrounds: The application automatically updates the background theme based on the current weather condition (e.g., clear, rain, or clouds) to provide an immersive experience.

Glassmorphism UI: A visually appealing, modern aesthetic using CSS backdrop filters.

Responsive Design: Fully optimized for mobile and desktop screens.

🛠️ Tech Stack
Frontend: HTML5, Tailwind CSS, JavaScript (ES6+).

API: OpenWeatherMap API.

Design: Custom Glassmorphism patterns.
